# Vue 3 + Vite

# This Todo App is built using Vue 3, Pinia, Quasar, and MongoDB.

![Todo Application](Frontend/src/assets/TodoApplication.png)


## Run backend
- > Open Terminal Window (power shell)
- > cd backend
- > npm install
- > node server.js

## Run frontend
- > Open Terminal Window (power shell)
- > cd frontend
- > npm install
- > npn run dev
